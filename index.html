My Challenges

#Speech Quiz

Speech synthesis, text to speech

This is a simple speech quiz that uses the speech synthesis API to read out the questions. The user can say the correct answer and the quiz will display it on the screen if it's right or wrong.

#Track your body

Handpose

This is a simple handpose balloon game that tracks if your forefinger touched/popped a balloon. Try the challenge of aiming your finger precisely to burst those balloons

#Foto herkenningsapp

MobileNet

This is a photo recognition app that uses the MobileNet model to identify objects in uploaded images. You make a photo of an object and the model will idetify the object and try to recognize what it is.

#Device orientation & motion

Deviceorientation, handleorientation

In this challenge I made a cocktail shaker where you need to shake for 30 seconds. Using deviceorientation, the game tracks your movements in real-time. Can you beat the clock and achieve the highest shake count?

#City Game

Geolocation API, Leaflet

This is a simple city game that uses the Geolocation API and Leaflet to display a map of the user's current location. The user can click on the map to guess the name of the city and the app will display the correct answer.
