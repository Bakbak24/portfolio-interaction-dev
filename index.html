My Challenges

#Speech Quiz

Speech synthesis, text to speech

This is a simple speech quiz that uses the speech synthesis API to read out the questions and answers. The user can select the correct answer and the quiz will read out the correct answer if the user gets it wrong.

#Track your body

Handpose

This is a simple handpose app that tracks your body and displays the body parts that are detected. The user can select the correct answer and the quiz will read out the correct answer if the user gets it wrong.

#Foto herkenningsapp

MobileNet

This is a photo recognition app that uses the MobileNet model to identify objects in uploaded images. MobileNet is a pre-trained deep learning model designed for mobile and embedded devices, making it efficient and fast for image recognition tasks.

#Device orientation & motion

Deviceorientation, handleorientation

This is a simple device orientation and motion app that uses the deviceorientation and devicemotion events to track the orientation and motion of the device. The app displays the alpha, beta, and gamma values of the device and updates in real-time as the device is moved.

#City Game

Geolocation API, Leaflet

This is a simple city game that uses the Geolocation API and Leaflet to display a map of the user's current location. The user can click on the map to guess the name of the city and the app will display the correct answer.
